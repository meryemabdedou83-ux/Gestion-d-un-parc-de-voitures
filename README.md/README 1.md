\# Gestion d’un parc de voitures



\## Description

Ce projet est réalisé en Python. 

Il permet de gérer un parc de voitures.



\## Classes

Voiture : représente une voiture avec sa marque, son modèle et son année.



Parc : représente un parc qui contient plusieurs voitures.



\## Fonctionnalités

\- Ajouter une voiture

\- Supprimer une voiture

\- Afficher les voitures
