POO Atelier 3 – Gestion d’un parc de voitures



Cet atelier consiste à créer un programme en Python pour gérer un parc de voitures.

Le système permet d’ajouter des voitures dans le parc, de retirer une voiture et de calculer le nombre de places disponibles.
